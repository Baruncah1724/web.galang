Cara Membuat Website dengan HTML dan CSS

Di zaman serba digital seperti sekarang, memiliki website sendiri sudah menjadi kebutuhan, baik untuk pribadi, bisnis, portofolio, blog, maupun toko online. Tapi bagaimana caranya membuat website dari nol? Apakah harus jago nge coding? Jawabannya: tidak.

kalau kamu seorang pemula seperti saya, kamu bisa mulai membuat website sederhana hanya dengan dua teknologi dasar: HTML dan CSS. Artikel ini akan membahas secara lengkap dan sistematis bagaimana sih membuat website sendiri dari awal, mulai dari konsep, penulisan kode, hingga melihat hasilnya langsung di browser.

Apa Itu HTML dan CSS?

1. HTML (HyperText Markup Language)

HTML adalah bahasa markup yang digunakan untuk membuat struktur dasar sebuah halaman web. Dengan HTML, kamu bisa menampilkan berbagai elemen seperti:

  • Teks dan paragraf
  • Gambar dan video
  • Tombol dan link
  • Formulir
  • Tabel

HTML adalah kerangka utama dari sebuah halaman web. Ibarat kata sebuah rumah, HTML adalah tembok, atap, dan jendela.

2. CSS (Cascading Style Sheets)

CSS adalah bahasa yang digunakan untuk mengatur tampilan atau desain dari elemen HTML. Dengan CSS, kamu bisa mengatur:

  • Warna latar belakang dan teks
  • Ukuran dan jenis font
  • Posisi dan ukuran elemen
  • Animasi dan transisi
  • Layout halaman secara keseluruhan

CSS adalah cat, dekorasi, dan furnitur dari rumah tersebut. Tanpa CSS, halaman web akan terlihat polos dan kaku, contoh besarnya ialah website saya ini tanpa menggunakan CSS website saya ini hanya berisikan kan kata tanpa background warna, background photo dan juga fitur tampilkan lainnya yg pastinya membuat sebuah tampilan tidak menarik.

Langkah-langkah Membuat Website Sederhana

1. Persiapkan Alat

Kamu hanya perlu:

  • Text Editor seperti Visual Studio Code atau stackbliz sama seperti yg saya gunakan
  • Browser seperti Google Chrome atau Firefox

2. Buat File HTML

Buat file bernama index.html dan isi dengan:

<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Website Sederhana</title>
</head>
<body>
  <h1>Selamat Datang di Website Saya</h1>
  <p>Ini adalah website pertama saya yang dibuat dengan HTML dan CSS.</p>
</body>
</html>

3. Tambahkan CSS

Buat file style.css lalu isi dengan:

body {
  background-color: #black;
  font-family: Arial, sans-serif;
  padding: 20px;
}

h1 {
  color: #2c3e50;
  text-align: center;
}

p {
  color: #555;
  font-size: 18px;
  line-height: 1.6;
}

Ingat jangan lupa hubungkan ke HTML dengan menambahkan ini di bagian <head>:

<link rel="stylesheet" href="style.css">

4. Lihat Hasilnya

Simpan kedua file di satu folder, lalu klik dua kali file index.html untuk membuka di browser. Selamat! Website pertamamu sudah jadi!

Menambahkan Elemen Lain

  • Gambar:
    <img src="gambar.jpg" alt="Gambar" width="300">

  <input type="submit" value="Kirim">
</form>

Tips Tambahan

  • Gunakan class dan id di HTML
  • Gunakan komentar untuk memudahkan pemahaman kode
  • Latihan terus menerus agar semakin mahir

Kesimpulan

Membuat website dengan HTML dan CSS adalah langkah awal yang penting dalam dunia web development. Kamu tidak perlu menjadi ahli dulu untuk membuat halaman sederhana. Yang penting adalah mulai mencoba dan terus belajar.

Jika kamu sudah paham dasar ini, langkah selanjutnya adalah belajar JavaScript, responsive design, dan cara mempublikasikan website ke internet.
